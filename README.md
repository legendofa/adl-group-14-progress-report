# ADL Group 14: Project Report

This repository includes the LaTeX files for our midterm project report. You can compile this with your local LaTeX instance or copy them in Overleaf.

A version of the Pdf is built with each push and the Github Workflow. Take a look in the workflow history in Github where you can also download the according PdF to the commit.

The main file is `paper.tex`.

There we have the following sections (short instructions / hints added to the sections):

- Problem Definition
- Related Works
- Methodology
- Current Progress
- Next Steps
